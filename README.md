# deeplx-tr-node
deepl api on steroids

## Install
```bash
npm i deeplx-tr-node
```

## Use
```javascript
const deeplTranslate = require('deeplx-tr-node')

!async function (){
  console.log(await deeplTranslate({ text: 'test  this and \n\n that', toLang: 'zh', fromLang: 'en' }))
}()
// { alternatives: [], text: '测试这个和\n\n 那个' }
```
